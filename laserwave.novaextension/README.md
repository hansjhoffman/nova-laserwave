# LaserWave

A clean 80's synthwave / outrun inspired theme for Nova.

### LaserWave

![](https://nova.app/images/en/dark/editor.png)

### LaserWave High Contrast (WCAG AAA)

![](https://nova.app/images/en/light/editor.png)

## Currently supports:

- Typescript & Javascript
- JSX & TSX
- Markdown
- JSON
- Yaml
- Toml
- Makefile
- Shell
- HTML
- SCSS/CSS/SASS
- SQL
- INI
